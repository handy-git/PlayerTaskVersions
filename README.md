### 引入jar

* 第一步
```
<repositories>
    <repository>
        <id>handy-maven</id>
        <url>https://maven.handyplus.cn/repository/maven-public/</url>
    </repository>
</repositories>
```
* 第二步
```
<dependency>
  <groupId>com.handy.playertask</groupId>
  <artifactId>PlayerTask</artifactId>
  <version>2.0.3</version>
  <scope>provided</scope>
</dependency>
```
